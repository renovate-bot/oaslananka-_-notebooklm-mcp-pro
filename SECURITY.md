# Security Policy

## Supported Versions

Security updates are provided for the latest released minor version.

## Reporting a Vulnerability

Report vulnerabilities privately through GitHub Security Advisories for this
repository. Include the affected version, reproduction steps, impact, and any
known mitigations.

Do not open public issues for suspected vulnerabilities.

## Handling

Maintainers will acknowledge valid reports within 72 hours, triage severity, and
coordinate a fix and disclosure timeline based on risk.
