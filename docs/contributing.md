# Contributing

See the repository [CONTRIBUTING.md](https://github.com/oaslananka/notebooklm-mcp-pro/blob/main/CONTRIBUTING.md) for the full development guide.

Short version:

```bash
git clone https://github.com/oaslananka/notebooklm-mcp-pro
cd notebooklm-mcp-pro
make bootstrap
make lint
make typecheck
make test
make docs
```

Use Conventional Commits and include tests for behavior changes.
